# Organizing an event

You are more than welcome to organize an event in the server. In fact, we would like to encourage you to! Organizing an event is fairly easy, but requires you to be level 65 or higher on survival.rocks. To get started, run `/create event` in-game. You will then receive a link where you can fill out details about the event.

<img width="660" height="64" alt="image" src="https://github.com/user-attachments/assets/2730ef14-51ce-4741-a121-d84a623b905d" />

The following details have to be provided:
- Event name
- Server location
- Warp name or command to access
- Date and start time
- Duration
- Description
- Cover image

Then it's time to submit your request. Your request will be reviewed by an event manager. You will receive a private message from our bot to let you know if the event was accepted or rejected.

Most submitted events are accepted, as long as the event is well described and appropriate. Events can be about anything: from marriages to a flash sale for your company, or something creative. It is required to show up at your event and to lead it!

Accepted events are announced to the `#events` channel on Discord, and can then also be found on the website. A reminder about the event is always sent to everyone interested, an hour prior to the start of the event.

Good luck! We look forward to see your event.
